# Ecommerce Sales Dashboard (Power BI Project)

## 📊 Project Overview
This project is an Ecommerce Sales Dashboard created using Power BI to analyze sales performance, profit trends, and customer behavior. The dashboard provides interactive visualizations that help businesses understand their sales data and make better decisions.

---

## 🚀 Dashboard Features
- Total Sales Amount Analysis
- Total Quantity Sold
- Total Profit Overview
- Average Sales Calculation
- Sales by State
- Sales by Customer
- Profit by Month
- Quantity by Category
- Sales by Sub-Category
- Payment Mode Analysis (UPI, Debit Card, Credit Card, EMI)
- Interactive Filters (State, Quarter)

---

## 🛠 Tools Used
- Power BI
- Microsoft Excel
- Data Visualization
- Business Intelligence

---

## 📈 Key Insights
- Maharashtra has the highest sales among all states.
- Clothing category contributes the highest quantity sold.
- Printers generate the highest revenue among sub-categories.
- UPI is the most used payment method.
